# Apple UI 设计风格更新总结

## 已完成的设计更新

### 1. 颜色系统 (Color System)
- **背景色**: 从深蓝色调改为纯黑色系统 (#000000, #1c1c1e, #2c2c2e)
- **强调色**: 采用 iOS 系统颜色
  - 蓝色: #0a84ff (iOS 蓝)
  - 青色: #64d2ff
  - 绿色: #30d158 (iOS 绿)
  - 红色: #ff453a (iOS 红)
  - 橙色: #ff9f0a (iOS 橙)
- **文字颜色**: 使用 Apple 的透明度系统
  - 主文字: rgba(255, 255, 255, 0.92)
  - 次要文字: rgba(235, 235, 245, 0.6)
  - 三级文字: rgba(235, 235, 245, 0.3)

### 2. 毛玻璃效果 (Backdrop Filter)
- 所有面板使用 `backdrop-filter: saturate(180%) blur(20px)`
- 符合 macOS/iOS 的毛玻璃材质效果
- 增强层次感和现代感

### 3. 圆角和阴影 (Radius & Shadows)
- **圆角**: 8px, 12px, 16px, 20px (Apple 标准)
- **阴影**: 更柔和的多层阴影系统
  - 小阴影: 0 1px 3px rgba(0, 0, 0, 0.3)
  - 中阴影: 0 4px 12px rgba(0, 0, 0, 0.4)
  - 大阴影: 0 8px 24px rgba(0, 0, 0, 0.5)

### 4. 字体系统 (Typography)
- 使用 `-apple-system, BlinkMacSystemFont, 'SF Pro Display'`
- 字重: 400 (常规), 500 (中等), 600 (半粗), 700 (粗体)
- 字间距: -0.01em 到 -0.02em (Apple 风格紧凑)
- 字体平滑: `antialiased` 和 `grayscale`

### 5. 导航标签 (Segmented Control)
- 采用 iOS 分段控制器样式
- 激活状态: 半透明白色背景 + 微妙阴影
- 过渡动画: cubic-bezier(0.4, 0, 0.2, 1) 0.25s

### 6. 卡片和面板 (Cards & Panels)
- 背景: rgba(28, 28, 30, 0.72) 带毛玻璃
- 边框: rgba(255, 255, 255, 0.08)
- 悬停效果: 轻微放大 + 阴影增强
- 按压效果: scale(0.98)

### 7. 按钮样式 (Buttons)
- 主按钮: 纯色 iOS 蓝 (#0a84ff)
- 悬停: 变为青色 (#64d2ff) + scale(1.02)
- 按压: scale(0.98)
- 去除渐变，使用纯色

### 8. 输入框 (Input Fields)
- 背景: rgba(255, 255, 255, 0.05)
- 聚焦: 蓝色边框 + 3px 蓝色光晕
- 过渡: 0.3s cubic-bezier

### 9. 表格样式 (Tables)
- 表头: 半透明白色背景
- 悬停行: rgba(10, 132, 255, 0.08)
- 边框: 使用分隔线颜色 rgba(84, 84, 88, 0.65)
- 去除渐变和强烈阴影

### 10. 滚动条 (Scrollbars)
- 宽度: 6px (更细)
- 颜色: rgba(255, 255, 255, 0.15)
- 悬停: rgba(255, 255, 255, 0.25)
- 背景: 透明

### 11. 动画曲线 (Animation Curves)
- 统一使用 Apple 的缓动函数: `cubic-bezier(0.4, 0, 0.2, 1)`
- 时长: 0.3s (标准), 0.6s (较慢)

### 12. 布局调整 (Layout)
- 顶部导航高度: 64px → 52px (更紧凑)
- 侧边栏宽度: 380px → 340px, 520px → 480px
- 内边距: 统一减小 2-4px
